## 📋 Описание проекта

**Task Tracker** — REST API для управления задачами с применением AOP для:
- Логирования всех операций
- Мониторинга производительности
- Обработки исключений
- Проверки прав доступа
- Кэширования результатов

### Функциональность:
- ✅ CRUD задач
- ✅ Логирование всех действий (через AOP)
- ✅ Замер времени выполнения (через AOP)
- ✅ Обработка ошибок (через AOP)
- ✅ Проверка прав доступа (через AOP)
- ✅ Кэширование (через AOP)

### Endpoints:
```
GET    /api/tasks          - Все задачи
GET    /api/tasks/{id}     - Задача по ID
POST   /api/tasks          - Создать задачу
PUT    /api/tasks/{id}     - Обновить задачу
DELETE /api/tasks/{id}     - Удалить задачу
GET    /api/tasks/stats    - Статистика
```
